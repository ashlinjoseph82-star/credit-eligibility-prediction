# Credit Completion & Degree Eligibility Prediction System

This project predicts degree eligibility and graduation risk for university students
based on category-wise credit completion under a credit-based academic system.

The project focuses on building a production-ready machine learning pipeline including:
- SQL-based data storage
- Synthetic data generation
- Training and evaluation of traditional ML models
- Prediction-ready deployment
- Interactive dashboard
- Version-controlled development lifecycle
